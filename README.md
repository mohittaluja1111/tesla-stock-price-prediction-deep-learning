## 📌 Project Overview

This project focuses on predicting Tesla stock prices using Deep Learning models such as **SimpleRNN** and **LSTM**.  
The project performs complete Time-Series Forecasting using Tesla historical stock market data.

The application includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Time-Series Forecasting
- Deep Learning Model Building
- Model Evaluation & Comparison
- Streamlit Deployment

# 🎯 Problem Statement

The objective of this project is to predict Tesla stock prices using Deep Learning models trained on historical stock market data.

The project compares:

- SimpleRNN
- LSTM

to identify the best performing model for stock price forecasting.
